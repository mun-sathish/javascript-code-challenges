## Scenario 1: Centering Elements (Multiple Methods)

### Problem
Center an element both horizontally and vertically in its container.

### Solution 1: Flexbox (Modern & Recommended)
```html
<div class="flex-container">
  <div class="flex-item">Centered Content</div>
</div>
```

```css
.flex-container {
  display: flex;
  justify-content: center;  /* Horizontal centering */
  align-items: center;      /* Vertical centering */
  height: 100vh;
  background: #f0f0f0;
}

.flex-item {
  padding: 20px;
  background: #007bff;
  color: white;
  border-radius: 8px;
}
```

### Solution 2: CSS Grid
```css
.grid-container {
  display: grid;
  place-items: center;  /* Centers both axes */
  height: 100vh;
}
```

### Solution 3: Absolute Positioning + Transform
```css
.absolute-container {
  position: relative;
  height: 100vh;
}

.absolute-item {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

**When to use:**
- **Flexbox**: Best for single-axis or simple layouts
- **Grid**: Best for complex layouts
- **Absolute**: When you can't change display property

## Scenario 2: Responsive Grid Layout

### Problem
Create a responsive grid that adapts without media queries.

```html
<div class="responsive-grid">
  <div class="grid-card">Card 1</div>
  <div class="grid-card">Card 2</div>
  <div class="grid-card">Card 3</div>
</div>
```

```css
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1.5rem;
  padding: 1.5rem;
}

.grid-card {
  background: white;
  padding: 2rem;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  transition: transform 0.3s ease;
}

.grid-card:hover {
  transform: translateY(-4px);
}
```

**Key:** `auto-fit` collapses empty columns, `auto-fill` keeps them.

## Scenario 9: Dark Mode with CSS Variables

```html
<button id="themeToggle">Toggle Theme</button>
<div class="content">Content here</div>
```

```css
:root {
  --bg-primary: #ffffff;
  --text-primary: #212529;
  --accent: #007bff;
}

[data-theme="dark"] {
  --bg-primary: #1a1a1a;
  --text-primary: #ffffff;
  --accent: #4dabf7;
}

body {
  background: var(--bg-primary);
  color: var(--text-primary);
  transition: all 0.3s ease;
}

.content {
  background: var(--bg-primary);
  color: var(--text-primary);
}
```

```javascript
const toggle = document.getElementById('themeToggle');
const theme = localStorage.getItem('theme') || 'light';
document.documentElement.setAttribute('data-theme', theme);

toggle.addEventListener('click', () => {
  const current = document.documentElement.getAttribute('data-theme');
  const newTheme = current === 'light' ? 'dark' : 'light';
  document.documentElement.setAttribute('data-theme', newTheme);
  localStorage.setItem('theme', newTheme);
});
```

## üìê CSS Flexbox Complete Cheatsheet

### Container Properties

#### display: flex
```css
.container {
  display: flex;        /* Block-level flex container */
  display: inline-flex; /* Inline-level flex container */
}
```

#### flex-direction
```css
.container {
  flex-direction: row;            /* Default: left to right */
  flex-direction: row-reverse;    /* Right to left */
  flex-direction: column;         /* Top to bottom */
  flex-direction: column-reverse; /* Bottom to top */
}
```

#### flex-wrap
```css
.container {
  flex-wrap: nowrap;      /* Default: single line */
  flex-wrap: wrap;        /* Multi-line, top to bottom */
  flex-wrap: wrap-reverse; /* Multi-line, bottom to top */
}
```

#### flex-flow (shorthand)
```css
.container {
  flex-flow: row wrap;           /* direction + wrap */
  flex-flow: column nowrap;
  flex-flow: row-reverse wrap;
}
```

#### justify-content (Main Axis)
```css
.container {
  justify-content: flex-start;    /* Default: start of container */
  justify-content: flex-end;      /* End of container */
  justify-content: center;        /* Center of container */
  justify-content: space-between; /* Equal space between items */
  justify-content: space-around;  /* Equal space around items */
  justify-content: space-evenly;  /* Equal space between and around */
}
```

#### align-items (Cross Axis)
```css
.container {
  align-items: stretch;     /* Default: fill container height */
  align-items: flex-start;  /* Start of cross axis */
  align-items: flex-end;    /* End of cross axis */
  align-items: center;      /* Center of cross axis */
  align-items: baseline;    /* Align baselines */
}
```

#### align-content (Multi-line)
```css
.container {
  align-content: stretch;       /* Default: lines stretch */
  align-content: flex-start;    /* Lines at start */
  align-content: flex-end;      /* Lines at end */
  align-content: center;        /* Lines at center */
  align-content: space-between; /* Equal space between lines */
  align-content: space-around;  /* Equal space around lines */
  align-content: space-evenly;  /* Equal space everywhere */
}
```

#### gap
```css
.container {
  gap: 1rem;              /* Both row and column gap */
  gap: 1rem 2rem;         /* row-gap column-gap */
  row-gap: 1rem;          /* Gap between rows */
  column-gap: 2rem;       /* Gap between columns */
}
```

### Item Properties

#### order
```css
.item {
  order: 0;    /* Default: source order */
  order: 1;    /* Appears later */
  order: -1;   /* Appears earlier */
}
```

#### flex-grow
```css
.item {
  flex-grow: 0;  /* Default: don't grow */
  flex-grow: 1;  /* Grow to fill space */
  flex-grow: 2;  /* Grow twice as much */
}
```

#### flex-shrink
```css
.item {
  flex-shrink: 1;  /* Default: can shrink */
  flex-shrink: 0;  /* Don't shrink */
  flex-shrink: 2;  /* Shrink twice as much */
}
```

#### flex-basis
```css
.item {
  flex-basis: auto;    /* Default: based on content */
  flex-basis: 200px;   /* Fixed width/height */
  flex-basis: 50%;     /* Percentage of container */
  flex-basis: 0;       /* Ignore content size */
}
```

#### flex (shorthand)
```css
.item {
  flex: 1;              /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */
  flex: 0 1 auto;       /* Default: don't grow, can shrink, auto basis */
  flex: 1 0 200px;      /* grow, don't shrink, 200px basis */
  flex: 2;              /* Grow twice as much as flex: 1 */
}
```

#### align-self
```css
.item {
  align-self: auto;       /* Default: inherit from container */
  align-self: flex-start; /* Override container's align-items */
  align-self: flex-end;
  align-self: center;
  align-self: baseline;
  align-self: stretch;
}
```

### Common Flexbox Patterns

#### 1. Perfect Centering
```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

#### 2. Space Between Header
```css
.header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

#### 3. Equal Width Columns
```css
.container {
  display: flex;
}
.item {
  flex: 1; /* All items equal width */
}
```

#### 4. Sidebar Layout
```css
.container {
  display: flex;
}
.sidebar {
  flex: 0 0 250px; /* Fixed 250px width */
}
.main {
  flex: 1; /* Takes remaining space */
}
```

#### 5. Card Grid
```css
.container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}
.card {
  flex: 1 1 300px; /* Grow, shrink, min 300px */
}
```

#### 6. Sticky Footer
```css
body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}
main {
  flex: 1; /* Takes all available space */
}
```

## üÜö Flexbox vs Grid: When to Use

### Use Flexbox When:
- ‚úÖ One-dimensional layouts (row OR column)
- ‚úÖ Content-driven sizing
- ‚úÖ Navigation bars
- ‚úÖ Centering items
- ‚úÖ Distributing space between items
- ‚úÖ Aligning items in a single direction

### Use Grid When:
- ‚úÖ Two-dimensional layouts (rows AND columns)
- ‚úÖ Layout-driven sizing
- ‚úÖ Complex page layouts
- ‚úÖ Overlapping elements
- ‚úÖ Precise placement control
- ‚úÖ Responsive grids without media queries

### Can Use Both:
```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 1rem;
}

.card {
  display: flex;
  flex-direction: column;
  justify-content: space-between;
}
```

## üí° Pro Tips

### Flexbox Tips:
1. Use `flex: 1` for equal-width items
2. Use `gap` instead of margins for spacing
3. `flex-wrap: wrap` for responsive layouts
4. Combine `justify-content` and `align-items` for centering
5. Use `order` sparingly (affects accessibility)

### Grid Tips:
1. Use `fr` units for flexible sizing
2. `auto-fit` collapses empty columns, `auto-fill` keeps them
3. Use `minmax()` for responsive columns
4. Named areas make complex layouts readable
5. `grid-column: 1 / -1` spans full width
6. Use `place-items: center` for quick centering
7. Combine with Flexbox for nested layouts

### Common Gotchas:
- **Flexbox**: `flex: 1` = `flex: 1 1 0%` (not `auto`)
- **Grid**: `fr` units don't work with `minmax()` minimum
- **Both**: `gap` doesn't work in older browsers (use fallbacks)
- **Flexbox**: `align-content` only works with `flex-wrap: wrap`
- **Grid**: Implicit grid tracks use `grid-auto-rows/columns`
